# Warming-Up Dashboard

When SveltyCMS starts up, instead of showing 503 errors during the boot phase, users see an SSR-rendered warming-up dashboard with live service status.

## What It Shows

- Real-time status of Database, Auth, Cache, and Content services
- Auto-refreshes via server polling — transitions to the target page when `READY`
- Actionable feedback if a service fails to start

## Implementation

The warming-up page lives at `src/routes/warming-up/`. It is served during the `INITIALIZING` state phase, before the system transitions to `READY`.
