# Smart CMS Migration

> **The Smart Importer has been migrated to a standalone plugin: [Smart AI-Driven Migration Pro](/src/plugins/smart-importer/smart-importer).**

The migration functionality is now provided by the **Smart AI-Driven Migration Pro** plugin, located at `src/plugins/smart-importer/`. This plugin expands the original 5-format importer to support **36+ CMS platforms** including WordPress, Drupal, Contentful, Sanity, Ghost, Shopify, Webflow, Storyblok, Prismic, and many more.

## Migration Path

| Old (Core)                                   | New (Plugin)                                                      |
| :------------------------------------------- | :---------------------------------------------------------------- |
| `src/services/smart-importer.ts`             | `src/plugins/smart-importer/index.server.ts`                      |
| `src/components/admin/smart-importer.svelte` | `src/plugins/smart-importer/components/TransformationTree.svelte` |
| `/config/importer`                           | `/config/migration`                                               |
| 5 formats supported                          | 36+ platforms supported                                           |

## Quick Reference

The plugin is **optional** — the Migration tile only appears in the Config page when the smart-importer plugin is installed and enabled. To install:

1. Ensure `src/plugins/smart-importer/` exists with all required files
2. Restart the CMS — the plugin is auto-discovered
3. Navigate to **Config → Extensions** to verify it's enabled
4. The **Migration** tile appears in **Config** automatically
