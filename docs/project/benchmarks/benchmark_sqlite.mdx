## 🔬 How to Reproduce

**Single test (console-only):**

```bash
bun test tests/benchmarks/api-latency.test.ts
```

**Single test (record to this report):**

```bash
BENCHMARK_RECORD=1 bun test tests/benchmarks/api-latency.test.ts
```

**Full SQLite matrix:**

```bash
bun run scripts/benchmark-matrix/index.ts --sql
```

**History-only (no MDX mutation, useful in CI):**

```bash
BENCHMARK_RECORD=1 BENCHMARK_HISTORY_ONLY=1 bun test tests/benchmarks/auth-performance.test.ts
```
