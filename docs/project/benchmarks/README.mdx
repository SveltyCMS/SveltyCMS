## 🔬 Methodology

Our benchmarks focus on real-world scenarios:
- **REST/GraphQL Latency**: p95 response times.
- **Bulk Ingestion**: Entries per second.
- **Memory Stability**: RSS and Heap growth under sustained load.
- **ACID Integrity**: Transaction overhead and rollback safety.

For more details on our testing strategy, see [Testing Strategy](../../tests/strategy.mdx).
