## 🚀 Methodology

All benchmarks are executed using the integrated high-fidelity benchmark matrix runner. You can execute the full suite or target specific databases directly from the root of the project:

```bash
# Run the full benchmark matrix across all database engines and variants
bun run benchmark

# Target a specific database engine (e.g. SQLite, MongoDB, PostgreSQL, MariaDB)
bun run benchmark --db=sqlite
bun run benchmark --db=mongodb

# Skip Redis-backed caching variants to only benchmark raw database adapters
bun run benchmark --db=sqlite --skip-redis
```

For more details on the testing environment, see the [Project Infrastructure](../../architecture/database/core-infrastructure.mdx).
