# Database Performance Benchmarks

SveltyCMS delivers consistent, sub-millisecond performance across all supported database adapters thanks to our **Unified Caching Layer**.

## 📊 Benchmark Reports

Select a database engine to view detailed latency splits, client journey simulations, and stress audit results:

- [**SQLite (via Drizzle)**](./benchmark_sqlite.mdx) — Default for local and edge.
- [**PostgreSQL**](./benchmark_postgresql.mdx) — Enterprise scaling with JSONB.
- [**MariaDB**](./benchmark_mariadb.mdx) — High-concurrency relational pooling.
- [**MongoDB**](./benchmark_mongodb.mdx) — High-volume unstructured data.

## 🚀 Methodology

All benchmarks are executed using the integrated `bun run tests/benchmarks/database-performance.ts` harness.
For more details on the testing environment, see the [Project Infrastructure](../../architecture/database/core-infrastructure.mdx).
