## ü§ñ Detailed Plan: SveltyCMS MCP Server

The MCP Server is our most defensible feature, positioning SveltyCMS as the first CMS AI agents can operate natively.

**Core Stack:**

- **Storage:** `@lancedb/lancedb` (Native C++ bindings for blazing-fast, zero-copy vector access).
- **Embeddings:** `fastembed` (ONNX Runtime-based, in-process embeddings - no external API needed).
- **Validation:** `valibot` (The fastest, smallest TypeScript validator).
- **Runtime:** Node.js with Singleton pattern for persistent DB connections.

**Capabilities:**

1. **Resources:** Exposes local documentation (`AGENTS.md`, Roadmap) and Svelte 5/Skeleton v4 best practices.
2. **Prompts:** Standardized templates for generating widgets following the "Three Pillars Architecture".
3. **Tools:**
   - `add_roadmap_item`: Appends user/agent suggestions to the roadmap.
   - `search_knowledge_base`: Queries the local LanceDB vector database for project context.
