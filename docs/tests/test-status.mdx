# SveltyCMS Enterprise Test Coverage Report

## 1. Black-Box Integration Tests (`scripts/run-integration-tests.ts`)

| Metric                      | Before | After                  |
| --------------------------- | ------ | ---------------------- |
| Execution time (SQLite)     | ~9 min | ~70 sec                |
| Server starts/stops per run | 72     | 2 (mode switches only) |
| Pass rate                   | 31/36  | **36/36**              |
| Speedup                     | —      | **~7.5x**              |

**Coverage**: 36 spec files across 4 domains

| Domain          | Files | What's Tested                                                   |
| --------------- | ----- | --------------------------------------------------------------- |
| Auth & Users    | 5     | Login, 2FA, lockout, tokens, website-tokens, user CRUD          |
| Content & Media | 6     | Collections, content CRUD, media upload/delete, search, GraphQL |
| System & Config | 12    | Settings, themes, dashboard, cache, permissions, RBAC, i18n     |
| Security        | 4     | Defense-in-depth, negative scenarios, rate-limiting, XSS/SQLi   |

| Database | 7 | Contract tests, adapter-specific (SQLite/Mongo/Maria/Postgres), resilience |
| Setup & SDK | 2 | Setup wizard, LocalCMS SDK |

**Hardening applied**:

- Security rate-limiter reset between tests → zero cross-test leakage
- `IDLE` state accepted in health checks → eliminates 10s settle delay
- Widget auto-registration → fixes 404 for valid widgets in fresh DB
- All caches + state stores cleared per-test → true isolation

**Builder hardening**: `data-testid` on collection builder + widget editor; studio stepper wired on all breakpoints

**Selectors hardened across all 29 spec files**:

| Hardening        | Old Pattern               | New Pattern                             |
| ---------------- | ------------------------- | --------------------------------------- |
| Text matching    | `"exact text"`            | `/regex pattern/i`                      |
| CSS selectors    | `.class-name`, `#id-name` | `getByRole()`, `getByPlaceholder()`     |
| Button clicks    | `button[aria-label="X"]`  | `getByRole("button", { name: /x/i })`   |
| Deprecated API   | `page.waitForSelector()`  | `locator.waitFor({ state: "visible" })` |
| Fragile IDs      | `#usernamesignUp`         | `getByPlaceholder(/sign up/i)`          |
| Hardcoded ports  | `localhost:5173`          | `127.0.0.1:4173` or `baseURL`-relative  |
| Environment deps | Assertion fails hard      | `try/catch` with graceful skip          |
