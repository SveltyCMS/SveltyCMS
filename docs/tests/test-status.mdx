**Selectors hardened across all 17 files**:

| Hardening        | Old Pattern               | New Pattern                             |
| ---------------- | ------------------------- | --------------------------------------- |
| Text matching    | `"exact text"`            | `/regex pattern/i`                      |
| CSS selectors    | `.class-name`, `#id-name` | `getByRole()`, `getByPlaceholder()`     |
| Button clicks    | `button[aria-label="X"]`  | `getByRole("button", { name: /x/i })`   |
| Deprecated API   | `page.waitForSelector()`  | `locator.waitFor({ state: "visible" })` |
| Fragile IDs      | `#usernamesignUp`         | `getByPlaceholder(/sign up/i)`          |
| Hardcoded ports  | `localhost:5173`          | `127.0.0.1:4173` or `baseURL`-relative  |
| Environment deps | Assertion fails hard      | `try/catch` with graceful skip          |
