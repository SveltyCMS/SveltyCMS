# Collection Builder and Content Management E2E Tests (`/config/collectionbuilder`)

This document details the end-to-end test specifications and coverage checklist for the core schema modeling engine and content CRUD workflows.

## Coverage Checklist

- [x] Creating, modifying, and deleting collection schemas
- [x] Adding content entries utilizing various schema widgets
- [x] Smoke tests for content preview and lists
- [x] End-to-end journey tests for content creators
- [x] Tab-based editor navigation (Basic Setup, Widgets & Fields, Settings & Permissions)
- [ ] Drag-and-drop field sorting in collection builder - _Gaps identified, pending integration_

## Implementation Details

Collection tests evaluate widget lifecycle mapping (Input, RichText, Select, Relation, Repeater, Image). Playwright mock interceptors intercept local and server database calls to ensure isolated verification.
