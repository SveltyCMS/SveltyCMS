# Access Management (`/config/access-management`)

## Surface

| Piece                   | Role                                                        |
| ----------------------- | ----------------------------------------------------------- |
| `+page.server.ts`       | Admin-only load; roles from locals; permission catalog      |
| `+page.svelte`          | Tabs: Permissions, Roles, Admin, Website Tokens; save/reset |
| `permissions.svelte`    | Permission matrix vs roles                                  |
| `roles.svelte`          | Create/edit/delete/reorder roles                            |
| `admin-role.svelte`     | Admin role chrome                                           |
| `website-tokens.svelte` | Website API tokens                                          |
| `access.remote.ts`      | Token remote helpers                                        |
