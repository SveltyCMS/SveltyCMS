# Config Sync & Trash

## `/config/sync`

| Piece             | Role                                          |
| ----------------- | --------------------------------------------- |
| `+page.server.ts` | Admin gate                                    |
| `+page.svelte`    | Sync / Backups / Debug tabs, status, Sync All |

## `/config/trash`

| Piece             | Role                                           |
| ----------------- | ---------------------------------------------- |
| `+page.server.ts` | Admin gate                                     |
| `trash-api.ts`    | list + restore via `fetchApi` (CSRF automatic) |
| `+page.svelte`    | UI + showConfirm                               |
| Testing           | `seed-trash` / `purge-trash`                   |

## Control → test matrix

| Control                  | Unit                       | E2E                      |
