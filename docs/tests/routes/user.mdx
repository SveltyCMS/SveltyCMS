# User Profile and Management E2E Tests (`/user`)

This document details the end-to-end test specifications and coverage checklist for user accounts, profiles, credentials management, and admin CRUD actions.

## Coverage Checklist

- [x] Profile details updates
- [x] Password changes and security checks
- [x] User management CRUD (for admins)
- [x] API verification for user retrieval
- [ ] User self-deletion & account cleanup - _Gaps identified, pending integration_

## Implementation Details

User profiles are validated under different role definitions (admin, editor, developer). Tests include file upload verification for avatars, username modifications, and timing-safe password updates.
