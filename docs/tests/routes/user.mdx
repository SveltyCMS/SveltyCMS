# User Profile & Management — Coverage Plan (`/user`)

This document is the **source of truth** for reaching honest 100% behavioral coverage of
`src/routes/(app)/user`. Every UI control is mapped to unit, integration, and E2E cases.

> **Policy:** Soft-skip `return` on missing UI is forbidden for P0/P1 cases. Prefer
> `test.skip(condition, reason)` with a seeded fixture, or fail hard.
