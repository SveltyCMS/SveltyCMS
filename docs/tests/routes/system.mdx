# System Configuration and Management E2E Tests (`/config/*`)

This document details the end-to-end test specifications and coverage checklist for the system administration settings, permissions matrix, and role-based access.

## Coverage Checklist

- [x] Global system settings editing
- [x] Multilingual / i18n localization transitions
- [x] User role permissions editing (admin panel)
- [x] Role-Based Access Control validation for admin/editor/guest actions
- [ ] Database backup and restore operations - _Gaps identified, pending integration_

## Implementation Details

System tests cover database-agnostic settings serialization, i18n URL prefixes, cookie verification under different subdomains, and verify permission changes block requests with `403 Forbidden` for unauthorized actors.
