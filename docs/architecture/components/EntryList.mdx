# entry-list Component

The `entry-list` is the primary component for displaying collection data in a tabular format with enterprise features like server-side search, filtering, and intelligent preloading.

---

## Architecture

```mermaid
flowchart LR
    subgraph Server["Server (+page.server.ts)"]
        DB[(Database)]
        SC[Server Cache]
    end

    subgraph entry-list["entry-list Component"]
        direction TB
        TH[Table Header]

    end

        HP[preloadData]

    end

    DB --> SC
    SC --> entry-list
    TR -->|Hover| HP
    HP --> PC
```

---

## Features

| Feature                | Description                                   |
| ---------------------- | --------------------------------------------- |
| **Server-Side Search** | Real-time search with debounced queries       |
| **Field Filtering**    | Per-column filtering options                  |
| **Smart Pagination**   | Configurable page sizes (10, 25, 50, 100)     |
| **Hover Preloading**   | Connection-aware speculative loading          |
| **Bulk Selection**     | Multi-item selection with Shift+Click         |
| **Sortable Columns**   | Click-to-sort with ASC/DESC toggle            |
| **Status Display**     | Visual status badges (Published, Draft, etc.) |

## Data Flow

```mermaid
sequenceDiagram
    participant U as User
    participant EL as entry-list
    participant URL as URL State
    participant S as Server

    U->>EL: Change filter/search/page
    EL->>URL: Update URL params
    URL->>S: Request new data
    S-->>EL: Return entries + pagination
    EL->>U: Render updated table

    Note over EL,U: Hover Preloading
    U->>EL: Hover over row
    EL->>S: preloadData(entry URL)
    S-->>EL: Cache response
```

## Plugin Extensibility

The entry-list component can be extended through plugins using the **`dashboard` injection zone** for list-level enhancements.

### Plugin Integration Points

Plugins can enhance the entry-list with:

- **Custom Actions**: Bulk operations, export tools, workflow automation
- **Data Visualizations**: Charts, analytics overlays, trend indicators
- **Filtering Extensions**: Advanced search, saved filters, custom predicates
- **UI Enhancements**: Custom columns, row decorators, status badges

### Example: Bulk Export Plugin

```typescript
import type { Plugin } from '@src/plugins/types';
import BulkExportButton from './BulkExportButton.svelte';

export const exportPlugin: Plugin = {
	metadata: {
		id: 'bulk-exporter',
		name: 'Bulk Export',
		version: '1.0.0',
		enabled: true
	},
	ui: {
		slots: [
			{
				zone: 'dashboard',
				slot: 'list_actions',
				component: BulkExportButton,
				order: 50
			}
		]
	}
};
```
