# AIEnrichment Widget

<blockquote>
  **Automated content enrichment widget** that leverages AI models to generate metadata, summaries,
  translations, and SEO data in real-time.
</blockquote>

## Overview

The **AIEnrichment widget** is a powerful automation tool designed to reduce manual editorial work. It can "read" other fields in a collection and use AI to generate new content. It supports multiple backends, prioritizing local execution via **Ollama** for privacy and zero cost, while supporting enterprise APIs like OpenAI.

### Key Features

- ✅ **Context Awareness**: Can consume data from multiple fields to generate a single enrichment output.
- ✅ **Multi-Model Support**: Integrated with Ollama (Llama3, Mistral), OpenAI, and Anthropic.
- ✅ **Task Presets**: Built-in prompts for Summarization, Key Point extraction, Tone shifting, and Translation.
- ✅ **Live Preview**: Editors can review AI-generated content before applying it to the document.
- ✅ **Background Processing**: Handles long-running generative tasks without blocking the UI.
- ✅ **Privacy First**: Fully supports local inference (no data sent to the cloud).

## Basic Usage

### Auto-Summarization

```typescript
import AIEnrichment from "@src/widgets/custom/AIEnrichment";
import RichText from "@src/widgets/core/rich-text";

export const postsCollection = {
  name: "Posts",
  fields: [
    RichText({ label: "Content", db_fieldName: "content" }),

    AIEnrichment({
      label: "AI Summary",
      db_fieldName: "summary",
      sourceField: "content",
      task: "summarize",
      model: "llama3",
      maxTokens: 150,
    }),
  ],
};
```

---

## Configuration Properties

| Property      | Type     | Required | Default           | Description                                     |
| ------------- | -------- | -------- | ----------------- | ----------------------------------------------- |
| `sourceField` | `string` | ✅ Yes   | -                 | The field to analyze (e.g. `body`, `title`)     |
| `task`        | `string` | ❌ No    | `'summarize'`     | Task type: `summarize`, `keywords`, `translate` |
| `model`       | `string` | ❌ No    | `'ollama/llama3'` | The AI model identifier                         |
| `prompt`      | `string` | ❌ No    | -                 | Custom system prompt for the AI                 |
| `maxTokens`   | `number` | ❌ No    | `500`             | Limit on generated content length               |

## Performance Considerations

Generative AI can be slow. SveltyCMS optimizes this by:

1.  **Debounced Triggers**: Only running enrichment when the user requests it or after a significant pause.
2.  **Streaming**: Displaying the AI output as it is generated for immediate feedback.
3.  **Caching**: Storing generated results to avoid redundant model execution.
