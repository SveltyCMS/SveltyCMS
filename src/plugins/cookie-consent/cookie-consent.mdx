# Cookie Consent Manager Plugin

GDPR-compliant cookie consent banner with granular consent controls for analytics and marketing cookies. Includes server-side consent logging for audit trails and geo-IP filtering for jurisdiction-aware display.

> **Free Plugin** — Included with SveltyCMS. No license required.

---

## Features

| Feature           | Description                                                       |
| :---------------- | :---------------------------------------------------------------- |
| Cookie banner     | Configurable GDPR-compliant consent banner                        |
| Granular controls | Separate toggles for analytics, marketing, and functional cookies |
| Consent logging   | Server-side persistence of consent decisions for audit trails     |
| Geo-IP filtering  | Jurisdiction-aware display (only show in EU/EEA regions)          |
| Headless delivery | Cross-origin consent via `allowedOrigins` for decoupled frontends |
| Theme support     | Light, dark, and system theme modes                               |
| Customizable      | Position, delay, privacy policy URL, and close button options     |

---

## Configuration

| Setting             | Type    | Default           | Description                                        |
| :------------------ | :------ | :---------------- | :------------------------------------------------- |
| `position`          | Public  | `bottom-left`     | Banner position: `bottom`, `bottom-left`, `center` |
| `privacyPolicyUrl`  | Public  | `/privacy-policy` | Link to your privacy policy page                   |
| `showCloseButton`   | Public  | `false`           | Show a close/dismiss button                        |
| `delayMs`           | Public  | `800`             | Delay before showing the banner (ms)               |
| `theme`             | Public  | `system`          | Visual theme: `light`, `dark`, `system`            |
| `allowedOrigins`    | Private | `[]`              | Cross-origin domains for headless frontends        |
| `persistConsentLog` | Private | `true`            | Store consent decisions server-side                |
| `geoIpFiltering`    | Private | `false`           | Only show banner in GDPR-relevant jurisdictions    |
