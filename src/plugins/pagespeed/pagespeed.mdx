# Google PageSpeed Insights Plugin

**Status:** ✅ Available (v1.1.0)
**Category:** Performance

Monitors Core Web Vitals (LCP, FCP, CLS) and performance scores using the Google PageSpeed Insights API with built-in caching to minimize API quota consumption.

---

## Features

- **Performance scoring** — Lighthouse-based scores (0-100) per entry URL
- **Core Web Vitals** — LCP, FCP, and CLS metrics tracked per page
- **Entry list column** — Performance score visible in collection entry lists
- **One-click refresh** — "Refresh PageSpeed" action per entry
- **API caching** — Results cached to minimize Google API quota usage
- **SSR enrichment** — Scores computed server-side and attached to entries

---

## Configuration

| Key             | Type   | Visibility | Description                                            |
| --------------- | ------ | ---------- | ------------------------------------------------------ |
| `defaultDevice` | string | Public     | Test device: `mobile` or `desktop` (default: `mobile`) |
| `apiKeySource`  | string | Private    | API key source: `settings` or `env`                    |

### Environment Variables

```
PAGESPEED_API_KEY=your-google-api-key
```

---

## Usage

Enable the plugin in **Config → Plugins**. The performance score column appears automatically in your collection entry lists.

### Refresh a Single Entry

Click the **Refresh PageSpeed** action button on any entry to re-run the audit.

### View Metrics

The performance column shows:

- **Score** — color-coded (green ≥ 90, orange ≥ 50, red < 50)
- **FCP** — First Contentful Paint (ms)
- **LCP** — Largest Contentful Paint (ms)
- **CLS** — Cumulative Layout Shift

## Collections

| Collection                 | Purpose                               |
| -------------------------- | ------------------------------------- |
| `plugin_pagespeed_results` | Cached PageSpeed scores per entry URL |

---

## API Quota

Google PageSpeed Insights API has a free tier of 25,000 requests/day. The plugin caches results for 24 hours per URL to minimize consumption.
