# WebMCP Plugin

**Status:** ✅ Available
**Category:** AI / Integration

Exposes your CMS as a Model Context Protocol (MCP) server, enabling AI coding assistants to query collections, read schemas, and interact with content directly.

---

## Features

- **MCP server** — Standard Model Context Protocol endpoint
- **Collection introspection** — AI assistants can discover and query your content
- **Schema exposure** — Widget definitions and field types available to AI
- **Read-only by default** — Writes require explicit configuration
- **Tenant isolation** — Respects multi-tenant boundaries

---

## Usage

The MCP endpoint is available at:

```
POST /mcp
```

### Connect an AI Assistant

Configure your AI coding tool to connect to:

```
https://your-cms.com/mcp
```

The assistant can then:
- List all collections and their schemas
- Query entries with filters
- Read individual entries by ID
- (Optionally) Create and update entries

### Configuration

| Key | Type | Description |
|-----|------|-------------|
| `allowWrites` | boolean | Enable AI write access (default: false) |
| `exposedCollections` | string[] | Collections exposed to AI (empty = all) |
| `maxResults` | number | Max entries per query (default: 50) |
