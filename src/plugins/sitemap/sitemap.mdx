# Sitemap Generator Plugin

Automated XML sitemap generator with full i18n/hreflang support and automatic search engine indexing pings. Designed for headless frontend architectures with configurable domain mapping.

> **Free Plugin** — Included with SveltyCMS. No license required.

---

## Features

| Feature              | Description                                               |
| :------------------- | :-------------------------------------------------------- |
| XML sitemap          | Auto-generates standards-compliant XML sitemaps           |
| i18n/hreflang        | Multi-language support with proper `hreflang` annotations |
| Search engine pings  | Automatically notifies Google and Bing on content changes |
| Headless domain      | Configurable `frontendDomain` for decoupled frontend URLs |
| Robots.txt injection | Optionally auto-injects sitemap reference into robots.txt |
| Multi-tenant         | Per-tenant sitemap configuration                          |
| Collection scoping   | Include/exclude specific collections from the sitemap     |

---

## Configuration

| Setting                 | Type    | Default | Description                                             |
| :---------------------- | :------ | :------ | :------------------------------------------------------ |
| `pingBing`              | Public  | `true`  | Notify Bing on sitemap updates                          |
| `includeHreflang`       | Public  | `true`  | Include hreflang annotations for multi-language content |
| `frontendDomain`        | Private | `""`    | Base domain for sitemap URLs (headless frontend)        |
| `autoInjectToRobotsTxt` | Private | `true`  | Auto-add sitemap reference to `/robots.txt`             |
