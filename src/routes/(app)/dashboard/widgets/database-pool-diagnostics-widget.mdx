# Database Pool Diagnostics Widget

Real-time database connection pool statistics and health recommendations. Displays total, active, idle, and waiting connections with a utilization bar, health status badge, and actionable recommendations from the DatabaseResilience system.

---

## 💰 Licensing & Pricing

**Price: €9.99** — Freemium model with a **14-day free trial** (no key required).

### Free Tier (No License / Trial Active)

| Feature          | Description                                                  |
| :--------------- | :----------------------------------------------------------- |
| Live pool status | Total, active, idle, and waiting connection counts           |
| Health badge     | Healthy/degraded/critical status indicator                   |
| Utilization bar  | Visual pool utilization percentage with color coding         |
| Recommendations  | Actionable health recommendations from the resilience system |
| Auto-polling     | Data refreshes every 30 seconds                              |
| Trial period     | Full access for 14 days from first admin registration        |

### Premium Tier (Valid License Required)

| Feature                | Description                                            |
| :--------------------- | :----------------------------------------------------- |
| **Everything in Free** | All free tier features included                        |
| Historical tracking    | Connection pool metrics over time with trend charts    |
| Query analysis         | Slow query detection and execution time breakdowns     |
| Alerting               | Notifications on pool saturation or health degradation |
| Export reports         | CSV/JSON export of pool diagnostics history            |

### License Verification

The widget checks license status via the internal API on mount:

```
GET /api/system/license-status?type=dashboard&id=database-pool-diagnostics
```

Without a valid license after trial expiry, live pool status and recommendations remain visible. Historical tracking, query analysis, and alerting require an active license.
