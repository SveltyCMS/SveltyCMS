# Security Architecture
# - 4-Layer Defense-in-Depth (Middleware → Dispatcher → Handler → Page Action)
# - AI Bot Defense: 28 UA patterns, 45 honeypot routes, progressive tarpit
# - Zero-Bias CSPRNG: rejection-sampling token generation
# - Cross-Origin Isolation: COOP + COEP + CORP on all API responses
# - Argon2id password hashing (64MB), AES-256-GCM encryption
# - 280+ automated security tests, 134 regression tests
